Week 1: Class logistics , Data preparation & Linear Regression reviews

Learning Objectives

By the end of the week, learners will be able to:

  • Describe how to ensure your success in this class
  • Get to know your classmates
  • Describe what Self Regulated Learning means to you

How to prepare

  • Familiarize yourself with this website and required course materials.
  • Review the syllabus and HW0.
  • Read PMA6 Ch3 and ASCN Ch 1 (Before Wednesday)
  • Refresh on linear regression if needed (Before Fri) PMA6 Ch 8, ASCN Ch 7,9

Monday overview

  • Introduction to the instructor, class structure, materials, requirements, expectations and resources.

Wednesday overview

  • Group Quiz on class logistics
  • Create a Data preparation reference flowchart

Friday overview

  • Jump start - write down everything you know about LinReg. (LJ)
  • Work as a class to organize these notes in this google doc
  • Recap on the purpose of linear regression models, assumptions, interpretation of predictors.

Week 4: Logistic Regression

Learning Objectives

  • Build and interpret a Logistic regression model on binary data
  • Use measure of model fit to compare between models.

How to prepare

  • PMA6 CH12.1-12.8, ASCN Ch11.1-11.3

Monday overview

  • Group Quiz on Model Building
  • QFT/LJ: Non-continuous outcomes
  • Fitting and interpreting Logistic Regression models.

Wednesday overview

  • Odds Ratios are always the odds of an event for one group compared to another group.
  • Some measures of model fit for Logistic regression is slightly different than for linear regression.

Friday overview

  • Use logistic regression to classify observations into two groups.

Week 6: Missing Data: Identification, Impact, Imputation

Learning Objectives

  • Explain the effects of missing data.
  • List and define the different missing data mechanisms.
  • Explain the typical methods of handling missing data and the problems with each.
  • Explain the mathematical model behind two imputation methods

How to prepare

  • NA

Monday overview

  • Group quiz on Logistic Regression & Classification
  • QFT on Missing Data
  • Effects of non-response

Wednesday overview

  • What are the mechanisms in which data can be missing?
  • What strategies can we use to deal with missing data in an appropriate manner?
  • What are some methods for imputation?

Friday overview

  • Multiple imputation is the gold standard of how to analyze data with missing values.
  • We’ll explore how this algorithm works with some sample data.
